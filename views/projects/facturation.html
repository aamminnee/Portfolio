Gestion Financière

1 semaines 2026

Description du Projet

Le concept : Une application web intranet dédiée à la gestion commerciale, permettant de fluidifier le cycle de vente en transformant instantanément les devis validés en factures comptables.

Ce projet (SAE) se concentre sur l'intégrité des données financières. Il offre un tableau de bord centralisé pour suivre les états des commandes (Devis vs Facturé) et une interface de gestion pour le portefeuille clients (CRM léger).

En tant que Développeur Backend, j'ai mis en place une logique SQL avancée pour automatiser la numérotation et la datation des documents, déchargeant ainsi l'application PHP d'une partie de la logique métier critique.

Challenge Technique

Le point clé de ce projet réside dans l'automatisation via la base de données. J'ai implémenté des Triggers SQL complexes pour générer automatiquement les numéros de factures et les dates lors de la conversion d'un devis. De plus, le calcul des totaux HT se fait dynamiquement via des requêtes SQL optimisées (jointures et agrégations), garantissant que les montants affichés sont toujours synchronisés avec les lignes d'articles.

Outils & Langages

  • PHP (PDO)
  • MySQL (Triggers)
  • Bootstrap 5
  • Composer
  • Gestion SQL Avancée
